Cardputer Ollama Integration

Connect your M5Stack Cardputer to Ollama and chat with AI models directly from the device.

What it does:
      Sends text from the Cardputer to a local Ollama server
      Receives AI responses and displays them on screen
      Communicates over WiFi
      Lightweight and straightforward
Requirements:
    M5Stack Cardputer
    Computer running Ollama
    Same local network connection
Setup
    Install and run Ollama on your computer
    Start a model in Ollama
    Flash the Cardputer application
    Enter the server IP address in the device
    Start chatting!
